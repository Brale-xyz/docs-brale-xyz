<Intro
  heading="API Concepts Overview"
  subheading="An overview of key resources and concepts used throughout the Brale API docs."
/>

- [Accounts](/key-concepts/accounts) - Your customers
- [Addresses](/key-concepts/addresses) - On-chain wallet addresses that can receive or send stablecoins
- [Financial Institutions](/key-concepts/financial-institutions) - Financial accounts that can receive or send funds
- [Transfers](/key-concepts/transfers) - Convert into, out of, and between stablecoins and fiat
- [Automations](/key-concepts/automations) - A unique bank account number that will send deposited fiat funds to a pre-set on-chain wallet address
- [Idempotency](/key-concepts/idempotency) - Ensure each POST request is processed only once
