<Intro
  heading="API Concepts Overview"
  subheading="An overview of key resources and concepts used throughout the Brale API docs."
/>

- Accounts - Your customers
- Addresses - On-chain wallet addresses that can receive or send stablecoins
- Financial Institutions - Financial accounts that can receive or send funds
- Transfers - Convert into, out of, and between stablecoins and fiat
- Automations - A unique bank account number that will send deposited fiat funds to a pre-set on-chain wallet address
- Idempotency - Ensure each POST request is processed only once
