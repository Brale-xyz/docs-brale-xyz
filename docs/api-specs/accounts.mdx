# Accounts

Accounts are the top-level resource in the Brale API. Each account can have multiple addresses, transfers, and other associated resources.

## List Accounts

Retrieve a list of all accounts associated with your API credentials.

```bash
GET /accounts
```

### Example Request

```bash
curl --request GET \
  --url https://api.brale.xyz/accounts \
  --header 'Authorization: Bearer ${ACCESS_TOKEN}'
```

### Example Response

```json
{
  "accounts": [
    {
      "id": "2Js1YFqlfxgNqC2KTPEjrWIwKU7",
      "name": "My Business Account",
      "status": "active",
      "created": "2024-01-01T00:00:00Z"
    }
  ]
}
```

## Get Account

Retrieve details for a specific account.

```bash
GET /accounts/{account_id}
```

### Example Request

```bash
curl --request GET \
  --url https://api.brale.xyz/accounts/2Js1YFqlfxgNqC2KTPEjrWIwKU7 \
  --header 'Authorization: Bearer ${ACCESS_TOKEN}'
```

### Example Response

```json
{
  "id": "2Js1YFqlfxgNqC2KTPEjrWIwKU7",
  "name": "My Business Account",
  "status": "active",
  "created": "2024-01-01T00:00:00Z",
  "balances": {
    "USD": "1000.00",
    "SBC": "500.00"
  }
}
```

## Account Resources

Each account can have the following associated resources:

- Addresses
- Transfers
- Financial Institutions
- Webhooks
